# AI-Powered Solution for Assisting Visually Impaired Individuals  

This project is an AI-powered application designed to assist visually impaired individuals in perceiving and interacting with their surroundings. Built using **Streamlit**, **LangChain**, and **Google Generative AI**, it provides functionalities such as scene understanding, text-to-speech conversion, and obstacle detection to improve accessibility and quality of life.

### 1. **Real-Time Scene Understanding**  
- Generates descriptive textual output to interpret the content of an uploaded image.  
- Helps users understand their surroundings effectively.  

### 2. **Text-to-Speech Conversion**  
- Extracts text from uploaded images using OCR techniques.  
- Converts the text into audible speech for seamless content accessibility.  

### 3. **Object and Obstacle Detection**  
- Identifies objects and obstacles in an image.  
- Provides insights to enhance user safety and situational awareness.  

### 4. **Personalized Assistance for Daily Tasks**  
- Recognizes items, reads labels, and offers task-specific guidance.
